# Calculator

This program a is CLI based calculator for _basic_ arithmetic.

## Examples

### Addition

> ```
> $ java Calc add 2 2
> 4.0
> ```

### Subtraction

> ```
> $ java Calc sub 4 2
> 2.0
> ```

### Multiplication

> ```
> $ java Calc mul 2 3
> 6.0
> ```

### Division

> ```
> $ java Calc div 6 2
> 3.0
> ```

## Build

> ./build.sh

## Usage

> java Calc [subcommmand][operand] [operand]
